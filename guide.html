3. js/index.js에서 fetch로 컴포넌트 로드

index.js 파일에서 fetch()를 사용하여 해당 HTML 파일을 불러오고, innerHTML을 이용해 지정된 id에 컴포넌트를 삽입합니다.

예시:

// hrkim.html 불러오기
fetch("components/hrkim.html")
  .then((response) => response.text())
  .then((data) => (document.getElementById("hrkim").innerHTML = data));

🌟 버튼 스타일 지정

1. 메인 색상 버튼

예시:

<!-- 작은 버튼 (Fill - Main 색상) -->
<button class="btn-s btn-fill btn-fill-main">Small Fill Main</button>
 
<!-- 작은 버튼 (Outline - Main 색상) -->
<button class="btn-s btn-outline btn-outline-main">Small Outline Main</button>
 
<!-- 중간 버튼 (Fill - Main 색상) -->
<button class="btn-m btn-fill btn-fill-main">Medium Fill Main</button>
 
<!-- 중간 버튼 (Outline - Main 색상) -->
<button class="btn-m btn-outline btn-outline-main">Medium Outline Main</button>
 
<!-- 큰 버튼 (Fill - Main 색상) -->
<button class="btn-l btn-fill btn-fill-main">Large Fill Main</button>
 
<!-- 큰 버튼 (Outline - Main 색상) -->
<button class="btn-l btn-outline btn-outline-main">Large Outline Main</button>
